# دروس - منصة مراجعة الفلسفة

مرحباً بك في **دروس**، المنصة التعليمية المخصصة لمراجعة مادة الفلسفة لطلاب السنة الثانية باكالوريا (2Bac) في المغرب، وتحديداً لشعبة الآداب والعلوم الإنسانية.

## 🌟 حول الموقع
موقع **دروس** هو أداة تعليمية تهدف إلى تبسيط مراجعة مادة الفلسفة وتوفير المحتوى الضروري للنجاح في الامتحانات الوطنية. يركز الموقع على المجزوءات الأربع المقررة في البرنامج المغربي:
1. **الوضع البشري** (الشخص، الغير، التاريخ)
2. 2. **المعرفة** (النظرية والتجربة، العلوم الإنسانية، الحقيقة)
   3. 3. **السياسة** (الدولة، العنف، الحق والعدالة)
      4. 4. **الأخلاق** (الواجب، السعادة، الحرية)
        
         5. ## 📖 إرشادات الاستخدام
         6. - **التنقل:** استخدم القائمة الرئيسية للوصول إلى المجزوءات المختلفة.
            - - **المفاهيم:** كل مجزوءة تحتوي على 3 مفاهيم أساسية مشروحة بالتفصيل.
              - - **المراجعة:** يمكنك استخدام الموقع كمرجع سريع للمفاهيم والمواقف الفلسفية.
